## Problem Description

Banks manage thousands of accounts and transactions daily. Manual systems can lead to calculation errors, security issues, and slow service.
This project implements a **Bank Management System (BMS)** that automates key banking operations such as account handling, deposits, withdrawals, and authentication.

---

## Objectives

* Accurate transaction processing
* Secure authentication
* Reliable data storage

---

## Features

* Create Account
* Login System
* Deposit Money
* Withdraw Money
* Check Balance
* File Handling using `.txt` files
